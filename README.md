# Jack - Agent Workspace Backup

This repository contains the complete workspace backup for **Jack**, a Clawdbot agent.

## What's Included

- **AGENTS.md** - Workspace rules and conventions
- **SOUL.md** - Core personality and behavior
- **USER.md** - Information about the user (Yash)
- **IDENTITY.md** - Agent identity details
- **TOOLS.md** - Local tool configurations
- **HEARTBEAT.md** - Periodic task configuration
- **BOOTSTRAP.md** - First-run setup script

## Restoration

To restore this agent:

1. Install Clawdbot: https://github.com/clawdbot/clawdbot
2. Copy these files to your Clawdbot workspace directory
3. Start a new session - Jack will wake up with all memories and personality intact
